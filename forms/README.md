# Формы опросника

HTML-формы для 6 типов клиентов барбершопа. Каждая форма — отдельная страница, залитая в Tilda как кастомный HTML-блок.

## Типы форм

| Файл | Тип клиента | Описание |
|------|-------------|----------|
| `hard.html` | `hard` | Для друзей барбершопа с тегом `hardcore` в yclients |
| `loyal.html` | `loyal` | Для постоянных клиентов со скидкой — содержит персональные факты |
| `client.html` | `client` | Для постоянных клиентов без скидки |
| `newbie.html` | `newbie` | Для тех, кто ещё не был или только записался |
| `accomp.html` | `accomp` | Для сопровождающих (не стригутся сами) |
| `regular_anon.html` | `regular_anon` | Для тех, кого нет в CRM |

Файлы находятся в корне репозитория.

## Технологии

- **Vanilla JS** — без фреймворков, минимальные зависимости
- **CSS scroll-snap** — пагинация вопросов горизонтальным свайпом
- **Tilda CDN** — хостинг изображений
- Данные из `functions/identify` приходят как GET-параметры в URL: `?name=Иван&master=Костя&visits=24`

## Классификация и персонализация

Форма подбирается в `functions/identify/classifier.py`. Приоритет:

1. Тег `hardcore` → `hard`
2. Нет визитов → `newbie`
3. Скидка > 0 → `loyal`
4. Найден, без скидки → `client`
5. Не найден → `regular_anon`

Для `loyal` и `client` в URL подставляются факты: имя мастера, количество визитов, часы в кресле, сэкономленные деньги, вес волос.

## Про стиль `hard.html`

Форма для `hardcore`-сектантов намеренно использует провокационный тон (особое обращение, грубый стиль шуток) — это осознанный UX-эксперимент для конкретной аудитории. Тег `hardcore` ставится вручную в yclients и означает «свой человек, с ним можно по-другому».
